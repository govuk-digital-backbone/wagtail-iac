# wagtail Infrastructure as Code Module

This module deploys a wagtail application to an existing Amazon Elastic Container Service cluster (expected to be managed by the [terraform-ecs-module](https://github.com/govuk-digital-backbone/terraform-ecs-module)).

This modules is instantiated by the private `wagtail-instances` repository which provisions and manages instances.

## What is wagtail?

wagtail is an open source project tracking tool that uses Kanban boards to help teams manage their work.

You can find more information about wagtail on their [GitHub repository](https://github.com/wagtailnban/wagtail).
